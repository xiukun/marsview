# 组件拖拽

打开左侧组件物料，可以直接**点击**或者拖拽组件到画布中，我们推荐点击的方式添加组件到画布。

![访问端工作台](/page/drag.png)

## 组件特性

- 容器组件支持无限嵌套。
- 表单项组件必须放在 Form 组件或者 SearchForm 组件中。
- 组件通过事件流可以完成联动、显隐等操作。
- 组件支持上下、左右排序。
- 组件支持复制和删除。例如：直接 ctrl+c 复制，ctrl+v 粘贴。

## 组件排序

组件支持排序，选中画布中的组件，会有一个工具条，点击上移或者下移，即可调整组件的位置。
